# AI News Agent

An automated AI-powered newsletter system that fetches the latest AI news, summarizes it using AI, and sends a styled HTML email every morning at 6 AM.

## Features

- Fetches top 5 AI news articles using RSS
- Generates AI summaries using Groq API
- Sends HTML email newsletters
- Fully automated using GitHub Actions
- Works even when PC is OFF

## Tech Stack

- Python
- RSS Feed
- Groq API
- Gmail SMTP
- GitHub Actions
